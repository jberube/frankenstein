## requirements

- nodejs
- npm
- zombie.js
- mocha
- ExpressJs

To install nodejs and npm on Ubuntu, you can try:
```make
make node
```

To install the packages this project depends on:
```make
make packages
```

To install both:
```make
make deps
```

To run the tests:
```make
make test
```
